# Student-management-system
A simple Student Management System built using Python. Supports add, view, search, update, and delete operations.

This is a simple console-based Student Management System built using Python.

## Features
- Add new students
- View all students
- Search student by roll number
- Update student details
- Delete student records

## Technologies Used
- Python
- File handling
